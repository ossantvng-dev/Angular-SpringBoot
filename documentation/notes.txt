Angular CLI

Create a new app
	
	ng new angular-app
	
Create new component

	ng g component counter
	
	ng g c components/invoiceList
	
Create a new Service
	
	ng g service services/item
	
	
Install ngrx
	
	ng add @ngrx/store@latest
	
Install dev tools
	
	Redux DevTools in your browser.
	
	ng add @ngrx/store-devtools@latest
	
Install effects
	
	ng add @ngrx/effects@latest
